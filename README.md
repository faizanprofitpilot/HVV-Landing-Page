# Hunza Valley Ventures, LLC Website

A modern, minimal, professional website for Hunza Valley Ventures, LLC.

## Setup

1. **Add your logo**: Place your logo file in the `assets/` folder and name it `logo.svg` (or update the filename in `index.html` if using a different format).

2. **Open the website**: Simply open `index.html` in your web browser, or serve it using a local web server.

## Design Features

- Clean, professional layout with generous spacing
- Responsive design that works on all devices
- Subtle hover effects for interactive elements
- Healthcare-adjacent, corporate aesthetic
- Designed to scale as more portfolio companies are added
